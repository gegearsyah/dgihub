## Design Principles

### Zenius-Style Clean Interface
- Minimal, uncluttered design
- Generous white space
- Clear typography hierarchy
- Subtle animations
- Focus on content

### Mobile-First
- Touch-friendly targets
- Thumb-zone navigation
- Progressive disclosure
- Bottom navigation
- Responsive design

### Bilingual Support
- Bahasa Indonesia (primary)
- English (secondary)
- Language switcher
- RTL support ready

## Quick Reference

### Color Palette

**Light Mode:**
- Primary: `#2563EB`
- Background: `#FFFFFF`
- Text: `#111827`

**Dark Mode:**
- Primary: `#3B82F6`
- Background: `#111827`
- Text: `#F9FAFB`

### Typography Scale
- H1: 32px
- H2: 24px
- H3: 20px
- Body: 16px
- Caption: 14px

### Spacing
- Base unit: 4px
- Card padding: 16px
- Section spacing: 24px

### Touch Targets
- Minimum: 44x44px
- Preferred: 48x48px
- Bottom nav: 56x56px
